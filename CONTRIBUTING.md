# Contributing

Thank you for your interest in contributing to the Xiaomi 12T (plato) device tree.

This project is maintained in my spare time as a hobby. Contributions are welcome and appreciated.

## Before Submitting Changes

Please ensure that:

* The change is relevant to Xiaomi 12T (plato)
* The change has been tested on actual hardware whenever possible
* The change does not introduce regressions
* Build errors and warnings have been reviewed

## Pull Requests

When submitting a pull request:

* Clearly describe the purpose of the change
* Keep commits focused and easy to review
* Reference related issues when applicable
* Avoid unrelated cleanup in the same pull request

Examples of acceptable contributions:

* Device-specific fixes
* Stability improvements
* Recovery improvements
* Power management improvements
* Configuration cleanup
* Android version compatibility fixes

## Bug Reports

When reporting a problem, please include:

* Build date
* Reproduction steps
* Relevant logs (logcat, dmesg, tombstones, etc.)
* Whether the issue is reproducible

Issues without sufficient information may be difficult to investigate.

## Notes

Some issues may originate from proprietary vendor components or platform limitations and may not be fixable within this repository.
